Want to make a glitter text effect in just a few steps? I will show you 3 easy ways to create glitter text effect:<\/p>\n

    \n
  • Online glitter text effect<\/a> using Textturizer<\/li>\n
  • Online glitter text<\/a> mockup using MockoFun<\/a><\/li>\n
  • Glitter text effect using Photoshop (check out PSDDude’s glitter text effect tutorial<\/a> or the free glitter patterns<\/a> from PhotoshopSupply)<\/li>\n<\/ul>\n

    Let’s see how!<\/p>\n

    1. How to Make Glitter Text Online<\/h2>\n

    Textturizer is an online text effect generator. It comes with lots of presets and allows you to combine text and textures from the Textures4Photoshop database.<\/p>\n

    Go to the Textturizer preset: glitter text generator<\/a><\/p>\n

    Open it up and you will get a glitter text effect like this:<\/p>\n

    <\/p>\n

    Because Textturizer is an online text effect generator, you can easily change the text, font, front and back textures, colors, etc.<\/p>\n

    Its interface is very straight forwards, but if you want to know about all the features it offers, check out the full set of features and how to use Textturizer<\/a> in one of my previous posts.<\/p>\n

    Simply change the text with your own text. For example, if I key in “more glitter” here’s what I get:<\/p>\n

    <\/p>\n

    Glitter Font Generator<\/h3>\n

    I’ve recently had a client ask me whether I can make a glitter font for him.<\/p>\n

    He wanted to use glitter fonts for Photoshop to write wedding invitations. Without getting too technical, the reality is that fonts are monochromatic. There’s an ongoing project called Open Type Fonts that will eventually allow having full colored fonts, but that’s still not yet ready for the open world. So, until then, let me show you the next best thing to glitter fonts.<\/p>\n

    Textturizer lets you choose from a variety of fonts and applies texture to it. So, you can use it as your own personal free online glitter font generator.<\/p>\n

    Here’s an example turning the Kavoon font into a glitter font:<\/p>\n

    <\/p>\n

    Gold Glitter Text Generator<\/h3>\n

    <\/p>\n

    I don’t need to give you exact values for my changes because I can simply grab the link from the address bar and share this exact result: Gold glitter text effect<\/a><\/p>\n

    To create a gold glitter text effect I’ve played around with the settings in Textturizer.<\/p>\n

    I switched to Bowlby One fonts, I decreased the saturation and increased the brightness. I also changed the background textures.<\/p>\n

     <\/p>\n

    Silver Glitter Text<\/h3>\n

    <\/p>\n

    Here’s the link to the online silver glitter text effect<\/a> made with Textturizer.<\/p>\n

    Starting from the gold text effect above, I simply changed the text.<\/p>\n

    Then lowered the saturation and changed the hue to give it a bit of a blue tint.<\/p>\n

    Using the search bar in Textturizer you can search through the Textures4Photoshop free textures<\/a> database and you can easily add glitter text to any photo.<\/p>\n

    How to Make Glitter Letters<\/h3>\n

    If we can make text with Textturizer, we can also make glitter letters. For this we have 2 options:<\/p>\n

      \n
    1. Make large glitter letters PNG images<\/li>\n
    2. Make alphabet glitter letters<\/li>\n<\/ol>\n

      Single Glitter Letters PNG<\/strong><\/p>\n

      <\/p>\n

      Go to this link in Textturizer to get the glitter letters PNG<\/a> files.<\/p>\n

      Let’s place in the text in Textturizer a single letter (“A”) and then change the text size to match the glitter letter PNG size you want to create. When making PNG images with Textturizer we simply need to turn off the background texture by un-checking the background checkbox.<\/p>\n

      I’ve used the Kavoon font as I like the way it looks. It looks like a diy glitter letter.\u00a0Put in B, C, D,… etc. and download each glitter letter as a PNG file with transparent background.<\/p>\n

      If you play around with the texture hue, saturation and brightness you can easily make sliver glitter letters or other colored glitter letters.<\/p>\n

      How to Make Alphabet Glitter Letters<\/strong><\/p>\n

      <\/p>\n

      Using the same technique as above, make the font size smaller and input the alphabet letters. You can download the glitter alphabet PNG<\/a> directly from Textturizer.<\/p>\n

      So, that’s how you make glitter letters online!<\/p>\n

      2. Make Glitter Text Online Using A MockoFun Template<\/h2>\n

      MockoFun is an text design online application that makes it very easy to create cool text effects. It’s a bit more advanced than Textturizer<\/a>, but far easier to use than Photoshop.<\/p>\n

      So, it’s an intermediate option for creating glitter text.<\/p>\n

      What you want to do is get a free account on MockoFun<\/a>. Then in the templates gallery you will find the Glitter Font<\/a> template, which is also free.<\/p>\n

      <\/p>\n

      Open the template up and customize it to your liking.<\/p>\n

      Double click to input your own text. You can create for example a glitter name tag or glitter logo.<\/p>\n

      Select the glitter text and move it around, rotate it or change the font size and font face. MockoFun has over 1500 built in fonts that you can use for free, but you can also install custom fonts<\/a>.<\/p>\n

      You can download the result as a JPG image, PNG glitter image with transparent background or even a 300dpi PDF printable glitter text design.<\/p>\n

      3. How to Make Glitter Text in Photoshop<\/h2>\n

      Of course, if you want more advanced glitter text effects you should probably use Photoshop.<\/p>\n

      I prefer Textturizer because you can create glitter text quick and easy, but Photoshop will give you more control.<\/p>\n

      If you are looking for more advanced glitter text effects check out PSDDude’s glitter text effect tutorial<\/a> or the free glitter patterns<\/a> from PhotoshopSupply.<\/p>\n

      Make a Glitter Text Using a Glitter Texture or Glitter Patterns<\/strong><\/p>\n

      Here on Textures4Photoshop you will find many glitter textures that are free for personal and commercial use (with attribution). For the above glitter text effects I used this seamless glitter texture<\/a>.<\/p>\n

      Go ahead, download the full size version and open it in Photoshop.<\/p>\n

      Add a new text layer and write the word “GLITTER”.<\/p>\n

      <\/p>\n

      Move the glitter layer in front of the text layer. Then right click on the glitter layer and choose “Create clipping mask”.<\/p>\n

      <\/p>\n

      The result should look like this:<\/p>\n

      <\/p>\n

      Alternatively you can use the free glitter patterns<\/a> from PhotoshopSupply. Double click on the glitter layer and in the layer styles use a Pattern Overlay with one of the glitter patterns.<\/p>\n
